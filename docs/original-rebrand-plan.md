## Stage 2: Build Rebranded Web App
**Skill**: vibecoding-webapp-swarm
**Goal**: Build a complete, rebranded AI video editing platform that improves on AutoEdit

### Key Innovations Over AutoEdit:
1. **Multi-platform support** (not just Premiere Pro - web-based + plugin ecosystem)
2. **Smarter AI clip selection** (context-aware, not just motion/shakiness detection)
3. **Free tier + fair pricing** (not $149 upfront)
4. **Support for all formats** (BRAW, MXF, ProRes, not just MP4/MOV)
5. **Multiple video types** (not limited to music videos)
6. **Better UX** (no plugin installation headaches, no security mode changes)
7. **Social listening dashboard** (built into the app)

### App Structure:
- Landing page (hero, features, pricing, testimonials)
- Dashboard (upload, AI editing tools, timeline preview)
- Social Listening Insights page
- Innovation/Comparison page
- Admin backend simulation

### Brand Identity:
- Name: **CutFlow AI**
- Tagline: "Where AI Meets Rhythm. Edit at the Speed of Thought."
- Color palette: Electric coral + deep obsidian + warm ivory
- Target: Music video editors, event creators, content creators
